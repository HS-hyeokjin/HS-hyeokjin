# 안녕하세요, 백엔드 개발자 김혁진입니다. 👋

> Java와 Spring Boot를 기반으로 서비스를 설계하고, AI 기능과 배포 환경까지 연결하는 백엔드 개발자를 지향합니다.


최근에는 문서 기반 AI 질의응답 서비스 **EVIDO**를 개발하며,
Spring Boot 기반 백엔드 서버와 FastAPI 기반 RAG 서버를 연동하고 있습니다.

## 🚀 Main Projects

### 📌 EVIDO - 문서 기반 AI 질의응답 서비스

> 사용자가 업로드한 문서를 기반으로 질문에 답변하고, 답변의 근거 문서 조각까지 제공하는 RAG 기반 AI 서비스입니다.

**주요 기능**

* 문서 업로드 및 문서 목록 관리
* 워크스페이스 기반 문서 관리
* 문서 기반 AI 질의응답
* 답변 근거 문서 조각 제공
* GPT 스타일 채팅 UI
* PDF / TXT 문서 뷰어
* JWT / OAuth2 기반 인증
* Docker, Jenkins, Nginx 기반 배포

**담당 및 경험**

* Spring Boot 기반 API 서버 설계
* 헥사고날 아키텍처 기반 도메인 분리
* FastAPI RAG 서버 연동
* Qdrant Vector DB 기반 문서 검색 구조 설계
* JWT / OAuth2 인증 흐름 구현
* React 프론트엔드와 API 연동
* EC2, Docker, Jenkins, Nginx 기반 배포 환경 구성

**Tech Stack**
`Java 21` `Spring Boot` `Spring Security` `JPA` `MySQL` `Redis` `React` `FastAPI` `Qdrant` `Docker` `Jenkins` `Nginx`

### 📌 Edututor - AI 기반 교육 플랫폼

* 교사 / 학생 회원 관리
* 수업 및 단원 관리
* 시험지 및 문제 관리
* 학생 답안 제출
* AI 기반 채점 및 학습 분석
* 학습 요약 및 맞춤형 문제 제공

**담당 및 경험**

* 팀장 역할 수행
* 로그인 / 회원가입 서비스 구현
* JWT 기반 인증 구조 설계
* Spring Security 기반 권한 처리
* Google, Naver OAuth2 로그인 구현
* AI 채점 및 학습 요약 기능 연동
* Blue-Green 배포 구조 설계 경험

**Tech Stack**
`Java` `Spring Boot` `Spring Security` `JPA` `JWT` `OAuth2` `React` `MySQL` `Docker`

* 결제 및 에스크로 서비스 담당
