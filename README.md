# 안녕하세요, 백엔드 개발자 김혁진입니다. 👋

> Java와 Spring Boot를 기반으로 서비스를 설계하고, AI 기능과 배포 환경까지 연결하는 백엔드 개발자를 지향합니다.


최근에는 문서 기반 AI 질의응답 서비스 **EVIDO**를 개발하며,
Spring Boot 기반 백엔드 서버와 FastAPI 기반 RAG 서버를 연동하고 있습니다.

---

## 🧑‍💻 About Me

* Java / Spring Boot 기반 백엔드 개발을 주로 공부하고 있습니다.
* JWT, Spring Security, OAuth2를 활용한 인증/인가 구현 경험이 있습니다.
* JPA를 활용한 도메인 설계와 데이터베이스 모델링에 관심이 많습니다.
* React 기반 프론트엔드와 연동되는 REST API 설계 경험이 있습니다.
* Docker, Jenkins, Nginx, AWS EC2를 활용한 배포 경험이 있습니다.
* 최근에는 RAG, Vector DB, LLM을 활용한 AI 서비스 개발에 관심을 두고 있습니다.
